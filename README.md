# Weather Dashboard

A simple weather dashboard project with a login page and a 3-day forecast.

## How to open

1. Open `Weather-Dashboard/index.html` in your browser.
2. Use the demo login below.

## Demo login

- Username: `admin`
- Password: `1234`

## Notes

- The dashboard loads weather data from WeatherAPI.
- The dashboard automatically loads Delhi weather on the dashboard page.
- If the app does not work from the file system, open it with a local server.

Enjoy the project! ☀️
